# Demo Guide

## Create Demo Room

1. Open Synapse Relay UI.
2. Click `Create Demo Room`.
3. Send a coding task.
4. Click `Start Panel Round`.

## Included Agents

- Mock Claude Coordinator
- Mock GPT Creative Strategist
- Mock DeepSeek Code Implementer
- Mock DeepSeek Skeptic Reviewer
- Mock Codex Executor Liaison

No API key is required for this flow.
